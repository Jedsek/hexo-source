# 安装GTK4
首先, 你得参照 [GTK官方页面](https://www.gtk.org/docs/installations/) 上的信息, 根据你的操作系统, 进行相应操作, 下载 GTK (版本是GTK4)  

本人的环境为 ArchLinux(WSL), 选择它是因为不需要过多的操心配置方面  
比如gtk的安装, ArchLinux直接一行命令解决:  

```bash
sudo pacman -S gtk4
```
